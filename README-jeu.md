###Morpion (Tic Tac Toe) – Python Tkinter

🎮 Interface jouable — 3 modes
🖥️ Langage : Python (Tkinter)

#Description

Ce projet est un jeu de Morpion (Tic Tac Toe) avec une interface graphique simple et élégante.

Modes de jeu
Joueur 1 vs Joueur 2 – Partie locale entre deux joueurs
Joueur vs IA (ML) – L’ordinateur joue avec une logique minimax simple
Joueur vs IA Hybride – IA plus stratégique

Le score des joueurs est affiché en haut de l’interface et se met à jour après chaque partie. Des popups stylés annoncent la victoire ou le match nul.


#Fonctionnalités  

Interface graphique fluide avec Tkinter
Choix entre 3 modes de jeu
Score en temps réel
Popups personnalisés pour victoire ou match nul
Boutons interactifs avec survol et curseur “main”
Retour au menu principal et bouton Exit


#Installation


Installez Python 3 si ce n’est pas déjà fait.
Clonez ou téléchargez le projet sur votre ordinateur.
Pas de dépendances externes nécessaires, Tkinter est inclus avec Python.

#Lancer le jeu

Ouvrez un terminal dans le dossier du projet.
Lancez le jeu avec la commande :

python main.py

Le menu principal s’affiche, choisissez votre mode de jeu.
Cliquez sur les cases pour jouer, le score est mis à jour automatiquement.
